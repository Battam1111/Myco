# Claude Code hooks system reference

Claude Code hooks reference (verified 2026-04-12):

SessionStart hook exists. Format in .claude/settings.json:
```json
{"hooks": {"SessionStart": [{"matcher": "", "hooks": [{"type": "command", "command": "myco hunger --execute"}]}]}}
```

30+ hook events available: SessionStart, SessionEnd, PreToolUse, PostToolUse, UserPromptSubmit, SubagentStart, FileChanged, PreCompact, PostCompact, etc.

Design decision: Myco uses CLAUDE.md Agent instructions for boot sequence (Agent-First), not SessionStart hooks (mechanical). Hooks are available as optional belt-AND-suspenders for users who want both.
