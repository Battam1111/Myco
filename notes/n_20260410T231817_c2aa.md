# Four-command minimum closed loop derivation

eat / digest / view / hunger came from three converging forces:
(a) A-Mem research: flat notes scale, trees don't → need a namespace-agnostic 'write' verb = eat.
(b) A14 internal contradiction: if notes can sit forever in 'raw' status, the substrate becomes a hoard, not a metabolism → need a lifecycle 'advance' verb = digest.
(c) Minimum closed loop test: remove any one of the four and something breaks. view without eat = empty. digest without view = blind. hunger without all three = meaningless. eat without hunger = unbounded growth.
The set is minimal and complete.
