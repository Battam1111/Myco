# excrete_reason is not optional

L10 rule: excreted notes must have a non-empty excrete_reason. Rationale: compression doctrine. If a note is deliberately removed from active rotation, the reason is the only thing future-you (or future-agent) will have to understand why. Excretion without reason is indistinguishable from forgetting, and the whole point of the digestive tract is that forgetting is a deliberate, traceable action.
