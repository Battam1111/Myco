# notes.py is the single source of runtime truth

Dependency rule: src/myco/notes.py is imported by cli (via notes_cmd.py), mcp_server.py, and src/myco/lint.py (L10 mirror). notes.py itself imports nothing from myco.cli or myco.mcp_server — one-way dependency. Scripts/lint_knowledge.py duplicates the validation logic so projects without the myco package can still lint. If the schema changes, both lint sites must update — L0 canon check does not (yet) enforce this, but the _canon.yaml → notes_schema is the declarative source both sites read.
