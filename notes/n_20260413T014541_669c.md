MCP Support Across All Major AI Tools (2026-04):

FULL MCP SUPPORT (stdio + remote): Claude Code, Cursor, Cline, Windsurf, Codex, VS Code+Copilot, Zed, Continue.dev, Replit
PARTIAL (remote only): ChatGPT API
NO MCP: ChatGPT (consumer), standard GPT

CONFIG FILES BY TOOL:
- Claude Code: .mcp.json (project root)
- Cursor: .cursor/mcp.json
- Cline: cline_mcp_settings.json
- Codex: ~/.codex/config.toml
- VS Code: .vscode/settings.json
- Windsurf: Settings UI
- Continue: .continue/mcpServers/
- Zed: settings.json context_servers

KEY FINDING: MCP is now universal across ALL major AI coding tools. Myco's .mcp.json works directly for Claude Code. For other tools, need per-tool config generation or a universal installer.

COWORK SPECIFIC: Desktop MCP servers bridge to Cowork via SDK proxy. Skills must be at .claude/skills/<name>/SKILL.md structure. Cowork has Linux sandbox with restricted PATH.
