# 仿生学全命名重构方案

## 决策

项目未公布前，将所有非仿生学命名改为仿生学命名。零妥协。

## 命名映射

| 当前 CLI/MCP | 仿生学名 | 真菌学对应 | 模块文件 |
|---|---|---|---|
| lint / myco_lint | immune / myco_immune | 免疫系统 | lint.py → immune.py |
| status / myco_status | pulse / myco_pulse | 生命脉搏 | (inline in mcp_server) |
| search / myco_search | sense / myco_sense | 感知 | (inline in mcp_server) |
| log / myco_log | trace / myco_trace | 生长年轮 | (inline in mcp_server) |
| view / myco_view | observe / myco_observe | 观察 | notes_cmd.py |
| init | seed | 播种/萌发 | init_cmd.py → seed_cmd.py |
| verify | diagnose | 健康诊断 | verify_cmd.py → diagnose_cmd.py |
| config | genome | 基因组 | config_cmd.py → genome_cmd.py |
| migrate | graft | 嫁接移植 | migrate.py → graft.py |
| import | assimilate | 同化吸收 | import_cmd.py → assimilate_cmd.py |
| inlet / myco_inlet | absorb / myco_absorb | 营养吸收 | inlet_cmd.py → absorb_cmd.py |
| graph / myco_graph | mycelium / myco_mycelium | 菌丝网络 | graph.py → mycelium.py |
| cohort / myco_cohort | colony / myco_colony | 菌落 | cohorts.py → colony.py |
| session / myco_session | memory / myco_memory | 记忆 | sessions.py → memory.py |
| compress / myco_compress | condense / myco_condense | 浓缩 | compress_cmd.py → condense_cmd.py |
| uncompress / myco_uncompress | expand / myco_expand | 扩展 | (same module) |
| propagate | propagate | 孢子传播（已仿生） | propagate_cmd.py |
| eat/digest/hunger/forage/prune/evolve/reflect | keep | 已仿生 | keep |

## 影响范围

- CLI: cli.py 所有 subparser 名称
- MCP: mcp_server.py 所有 @mcp.tool name
- 模块: src/myco/*.py 文件名（git mv）
- 测试: tests/unit/*.py 所有 import 和调用
- 文档: 全部 .md 文件中的命令引用
- _canon.yaml: 所有引用
- templates/: 所有模板文件
- README x3: 所有命令示例

## 执行策略

单独一个 session，全量执行 + lint --fix + 测试验证。不分阶段，一次到位。
