# Dogfooding rule: the kernel eats its own work first

Invariant: every design decision made in the Myco kernel must pass through the kernel's own notes/ tract before it enters wiki/ or MYCO.md. The kernel is not a special case — it is a Myco instance running the Myco framework. If the framework fails for the kernel, it will fail for every user. This note itself is proof-of-concept: the kernel just ate its own dogfood rule.
