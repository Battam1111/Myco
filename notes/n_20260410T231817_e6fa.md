# Flat notes scale, folder hierarchies don't

Research sources cited in the digestive debate:
- A-Mem (arXiv 2502.12110): atomic notes with metadata outperform hierarchical memories.
- MemOS (arXiv 2507.03724): three-tier memory with MemCube metadata; the value is in the metadata, not the folder.
- Obsidian Dataview: validated flat notes to 18k+ notes through query-based navigation.
- Voyager (arXiv 2305.16291): compositional executable skills, curriculum cold start — validates content over structure.
Consequence for Myco: notes/ is flat. Status is a frontmatter field, not a subdirectory.
