# Non-linear jumps are allowed in the lifecycle

Lifecycle diagram is raw → digesting → {extracted | integrated | excreted}, but non-linear jumps are explicitly allowed: raw → integrated directly (when the note is already mature), raw → excreted (when it turns out to be noise), extracted → integrated (when extraction is complete). The status is a property of the note right now, not a breadcrumb trail. digest_count tracks how many times the agent has touched it, which gives the temporal dimension for free.
