# hunger exits non-zero on concerning signals

myco hunger returns exit code 1 when any of raw_backlog, stale_raw, no_deep_digest, no_excretion trigger. This enables 'myco hunger || echo needs attention' in shell pipelines, and lets pre-commit hooks or CI gate on metabolic health. 'healthy' and 'promote_ready' alone return 0 — promotion opportunities are not failures.
