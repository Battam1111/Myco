# Three ASCC pain points → three architectural gaps

Yanjun's ASCC practice surfaced three pain points: (1) 没办法很好地将内容消化到 Myco 中 → 缺 digestive tract; (2) 没办法很好地使用 Myco → 缺 trigger layer; (3) 甚至意识不到要使用 Myco → 缺 reflex hook. These map to three architectural gaps: missing commons layer, narrative-only digestive tract, missing trigger mechanism. The four-command set addresses (1) and (2); the MCP tool descriptions with trigger-condition lists address (3).
