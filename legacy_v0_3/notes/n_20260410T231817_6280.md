# MCP agent assumption is now explicit

Decision: Myco does NOT try to run without an MCP-speaking agent. The README now states explicitly: 'Myco assumes you work with an agent that speaks MCP. Pure-human CLI usage works but loses the reflex layer.' This is a conscious scope boundary — building reflex capture without an agent would require a daemon/watcher layer that competes with the agent, and the whole point of Myco is to be the agent's OS, not a second brain running in parallel.
