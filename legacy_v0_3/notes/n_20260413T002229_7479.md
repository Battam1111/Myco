# Editable install confirmed as primary — src/*.py joins mycelium

Design decision confirmed: editable install (git clone + pip install -e) is the PRIMARY and ONLY recommended distribution path. PyPI is discovery/demo only.

Implications for mycelium: since everything is editable, the kernel code files (src/myco/*.py) are now also part of the living substrate. They must be connected to the mycelium network too — lint.py should reference the lint dimensions it implements in docs, notes.py should reference the hunger signals it computes, evolve.py should reference the skills it mutates. The mycelium network expands from "docs+notes+wiki+skills" to include "src/myco/*.py" as full citizens.
