# docs/primordia/ — Craft Decision Records

> **What is primordia?** This directory stores Myco's craft debate records (traditional-craft decision documents). Each craft is a multi-round adversarial debate that produces a high-confidence design decision. Crafts follow the protocol defined in `docs/agent_protocol.md`.
>
> **Lifecycle labels**: `ACTIVE` = still evolving | `SUPERSEDED` = replaced by a newer craft | `ARCHIVED` = compiled to formal docs, moved to `archive/`
>
> **Soft limit**: Active craft count targets <= 60 (Wave 36 re-baseline). When count exceeds this, the oldest COMPILED crafts are compressed into `archive/`.
